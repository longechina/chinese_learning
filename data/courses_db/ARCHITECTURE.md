# Courses Database (courses_db/)

## Purpose
Stores structured course content for Chinese, English, Hugging Face, and NLP modules.

## Structure
- `chinese/`, `english/`, `nlp/` : JSON files per chapter
- `hf_course/` : Chapters, notebooks, subtitles, and course assets

## Features
- Allows uploading new courses and editing existing ones
- Courses are linked to chapters for structured retrieval
- Supports multi-language content (Chinese/English)
