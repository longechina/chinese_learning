# Raw Materials Database (raw_materials/)

## Purpose
Stores unstructured materials from OCR or user uploads.

## Structure
- `audio/`, `docx/`, `excel/`, `images/`, `misc/`, `pdf/`, `py/`, `scripts/`, `video/`
- `raw_materials_index.json` : Index for searching
- `RAW_MATERIALS_UPLOAD_RULES.md` : Upload rules

## Features
- Independent resources, not linked to courses
- Searchable by tags and date
- Supports cloud-first access with local caching
