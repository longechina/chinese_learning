# Quiz Database (quiz_db/)

## Purpose
Stores quiz data for structured learning.

## Structure
- `associated/` : Quizzes linked to courses and chapters
- `ai_generated/` : AI-generated quizzes from templates
- `independent/` : Independent quizzes
- `quiz_index.json` : Index for searching quizzes
- `QUIZ_UPLOAD_RULES.md` : Rules for naming and uploading quizzes

## Features
- Supports multiple question types: multiple choice, fill-in-the-blank, Q&A, matching, programming
- Supports multimedia questions (images, audio, video)
- UI validation on upload
