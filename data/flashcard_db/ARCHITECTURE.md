# Flashcard Database (flashcard_db/)

## Purpose
Stores flashcard data for structured learning.

## Structure
- `chinese/`, `english/`, `nlp/` : Each folder has `ch_01`, `ch_02`, `ch_03` with `flashcards.json`

## Features
- User can upload flashcards per chapter
- Supports linking flashcards to courses and notes
- Allows fast retrieval for study sessions
