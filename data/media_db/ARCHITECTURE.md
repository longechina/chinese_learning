# Media Database (media_db/)

## Purpose
Stores all multimedia learning resources (images, video, audio, GIFs) and independent assets.

## Structure
- `associated/` : Media linked to specific courses
- `independent/` : Media not linked to courses (organized by type)
- `background.jpg` : UI background
- `media_index.json` : Index for search
- `MEDIA_UPLOAD_RULES.md` : Rules for naming and uploading media

## Features
- Supports look-and-speak, course illustrations, diagrams
- Indexed for fast search by course, type, and tag
- Supports cloud sync and local caching
